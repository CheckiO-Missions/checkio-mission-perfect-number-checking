You just need to join all previous hints, return the result of comparison sum of divisors and original number and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def is_perfect(n: int) -> bool:

    sum_divisors = 1
    for i in range(2, int(n**0.5) + 1):
        quot, rem = divmod(n, i)
        if not rem:
            sum_divisors += i
            if i != quot: 
                sum_divisors += quot
                
    return sum_divisors == n

Use generator with condition as argument of sum() function. 

def is_perfect(n: int) -> bool:

    return sum(i for i in range(1, n) if not n % i) == n

Using set simplify our code because set keeps only unique elements. 

def is_perfect(n: int) -> bool:

    divisors = {1}
    for i in range(2, int(n**0.5) + 1):
        quot, rem = divmod(n, i)
        if not rem:
            divisors |= {quot, i}

    return sum(divisors) == n
