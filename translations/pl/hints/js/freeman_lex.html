Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę zadeklarować zmienną dla sumy dzielników. Iterujemy po liczbach całkowitych aż do pierwiastka kwadratowego z podanej liczby włącznie i sprawdzamy, czy każda z nich jest dzielnikiem podanej liczby (reszta z dzielenia musi być równa 0). Jeśli tak - proszę dodać również iloraz (jeśli nie jest równy bieżącemu dzielnikowi).

Ok, od czego powinienem zacząć?

Zadeklarujmy zmienną dla sumy dzielników z wartością początkową 1, ponieważ każda liczba jest podzielna przez 1. 

let sum = 1;
Co zrobić dalej?

Używając for loop proszę iterować po liczbach całkowitych od 2 do pierwiastka kwadratowego z podanej liczby włącznie. Aby znaleźć całkowitą wartość pierwiastka kwadratowego, proszę użyć metody Math.sqrt(). 

for (let i = 2; i <= Math.sqrt(n); i++) {...}
Nadal potrzebuję podpowiedzi.

Teraz wewnątrz pętli trzeba obliczyć iloraz i resztę z dzielenia podanej liczby i bieżącej liczby całkowitej i. 

const quot = n / i, rem = n % i;
Co zrobić z tymi wartościami?

Cóż, jeśli reszta rem == 0 (lub !rem), oznacza to, że liczba n jest podzielna przez liczbę całkowitą i. Zatem liczba i jest dzielnikiem liczby n i należy dodać ją do sumy dzielników liczby sum. 

if (!rem) {
    sum += i;
    ...
}
Czy to wszystko?

W tym przypadku (not rem) musi Pan również dodać iloraz quot do sumy dzielników, ponieważ sprawdzamy nie wszystkie liczby całkowite aż do liczby, ale tylko do pierwiastka kwadratowego. Aby uniknąć dodawania tego samego dzielnika (gdy iloraz == i), proszę sprawdzić. 

if (i!== quot) sum += quot;
Jak powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić wynik porównania sumy dzielników i oryginalnej liczby oraz zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

function isPerfect(n: number): boolean {

    let sum = 1;  
    for (let i = 2; i <= Math.sqrt(n); i++) {
        const quot = n / i, rem = n % i;
        if (!rem) {
            sum += i;
            if (i!== quot) sum += quot;
        }
    }
    return sum === n;
}

Jasne, proszę pobrać!

Proszę przekonwertować liczbę na tablicę jej zakresu i zredukować ją, dodając odpowiednie dzielniki. 

function isPerfect(n: number): boolean {

    const sum = [...Array(n).keys()]
    .reduce((a, b) => a + (n%b == 0? b: 0), 0);

    return sum == n;
}
