以下是整个解决方案：

def is_perfect(n: int) -> bool:

    sum_divisors = 1
    for i in range(2, int(n**0.5) + 1):
        quot, rem = divmod(n, i)
        if not rem:
            sum_divisors += i
            if i!= quot: 
                sum_divisors += quot
                
    return sum_divisors == n

使用带有条件的生成器作为sum() 函数的参数。 

def is_perfect(n: int) -> bool:

    return sum(i for i in range(1, n) if not n % i) == n

使用集合可以简化代码，因为集合只保留唯一元素。 

def is_perfect(n: int) -> bool:

    divisors = {1}
    for i in range(2, int(n**0.5) + 1):
        quot, rem = divmod(n, i)
        if not rem:
            divisors |= {quot, i}

    return sum(divisors) == n
